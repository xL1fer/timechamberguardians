Server Rules 路 02/04/2018
1. Don't kill friendly N.P.C.'s such as Torch Grunt, Medic Grunt,etc ;

2. Don't block passages: doorways, ladders, vents,... ;

3. Don't abuse the command "rtv" because after a few times, it will become distracting to others ;

4. Don't speedrun through maps ;

5. Don't troll, anyone trolling will be kicked, or banned;

6. There are two exceptions to the rules, rule 1- you may kill N.P.C.'s if we don't need them anymore,
rule 4- you may speedrun if you're the only one in the server, or with permission of an admin.
